# Setup (NLW-11)

Esse é um porjeto que ajuda na organização de tarefas do seu dia-a-dia onde nela ira informa a sua tarefa e os dias da semana que ira fazer essa tarefa ajudando assim a se manter organizado.

## Tecnologias 
### Server 
* Fastify
* Typescript
* PrismaORM
### Web
* React
* Radix-ui
* Tailwindcss
* Typescript
* Vite
* Axios
### Mobile
* Expo
* Nativewind
* React-Native
* Tailwindcss
